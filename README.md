## 介绍

迪拉熊Bot是一个主要面向Maimai音游玩家群体的Python Bot，在基础的群聊Bot功能之上追加了模块化的各类游戏功能支持。

迪拉熊Bot基于NoneBot框架和OneBot v11协议，可以通过非常愉快的方式快速进行本地部署。

## 快速部署

```shell
git clone xxx.git

cd DXKuma-Bot

pip install -r requirements.txt

python start.py
```
